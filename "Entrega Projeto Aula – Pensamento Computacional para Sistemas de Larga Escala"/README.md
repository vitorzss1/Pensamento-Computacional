# Pensamento Computacional – Atividade de Decomposição (Profª. Kadidja Valeria)

## Descrição

Este repositório contém uma atividade acadêmica da disciplina de Pensamento Computacional, 
desenvolvida no curso de Engenharia de Software.

O projeto apresenta a aplicação prática dos conceitos de pensamento computacional por meio 
da decomposição detalhada do processo de compra de ingressos online. A proposta demonstra como 
tarefas do cotidiano podem ser organizadas em etapas menores para facilitar análise, entendimento 
e modelagem computacional.

## Objetivo da Atividade

Aplicar os princípios do pensamento computacional utilizando decomposição, abstração, reconhecimento 
de padrões e algoritmos para representar o fluxo completo de uma compra de ingresso online.

## Tema do Projeto

Compra de ingresso online utilizando plataformas digitais.

## Pensamento Computacional Aplicado

### Decomposição

O processo foi dividido em pequenas etapas sequenciais:

* Acesso ao celular.
* Desbloqueio do aparelho.
* Busca por plataformas de ingressos.
* Escolha do evento.
* Cadastro de dados pessoais.
* Geração do código de pagamento.
* Pagamento bancário.
* Confirmação da compra.
* Visualização do ingresso.

### Reconhecimento de Padrões

Foram identificados padrões comuns em:

* Plataformas de e-commerce.
* Sistemas digitais de pagamento.
* Aplicativos de autenticação e compra online.

### Abstração

O projeto considera apenas as ações essenciais necessárias para concluir a compra do ingresso, ignorando detalhes secundários.

### Algoritmos

Toda a sequência descrita representa um algoritmo lógico executado pelo usuário para atingir um objetivo específico.
